Bitonic Sort Implementation
This folder contains the implementation of the Bitonic Sort algorithm in different parallel computing frameworks. 
The Bitonic Sort is a well-known sorting algorithm that can be efficiently parallelized to improve performance on parallel architectures.

Files
bitonic_serial.cpp: Serial implementation of the Bitonic Sort algorithm in C++.
bitonic_omp.cpp: OpenMP implementation of the Bitonic Sort algorithm in C++.
bitonic_mpi.c: MPI implementation of the Bitonic Sort algorithm in C.

Usage
To compile and run the different implementations, simply run the run.sh file.
